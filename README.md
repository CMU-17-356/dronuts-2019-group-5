## Quickstart
```shell
## Setup
npm install
npm run build

## To run locally
# Local Node
npm start

# or, use Docker
docker-compose up

# or, watch for changes while developing
npm run watch

## To run tests
npm run test
```

## Tools used
- Language: Typescript (Node v8.0.0, npm 5.0.0)
- Framework: Express
- Linting: TSLint
- Testing: Jest
- Dependency Management: Greenkeeper
- CI: Travis CI
